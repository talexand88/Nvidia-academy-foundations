Findings (3)

Accessibility CTA fails WCAG contrast

The "Enroll Now" button uses white text on NVIDIA Green (#76B900). Contrast ratio is 2.41:1, below the AA minimum of 4.5:1 for normal text. Change to black text on green, or switch the background to brand-green-secondary (#004831) which passes white text at 11.6:1.

Cited: §01.1.1 · §02.7.2 · §04.2.4
Typography Title uses title case, not sentence case

"AI Infrastructure Fundamentals" uses Title Case (every major word capitalized). The Academy system requires sentence case across all surfaces — only the first word and proper nouns are capitalized. Change to "AI infrastructure fundamentals."

Cited: §01.3.5
Hero composition Correct hero variant for a course title

Embossed-glass NVIDIA hero composition is the canonical variant for course title cards. The linear-scratched variant is correctly reserved for unit and section openers.

Cited: §01.5.2 · §05.2.1

What the review covers

Color compliance

Only sanctioned hex values appear · CTA buttons use black-on-green · stripe colors match arch-status rules · note callouts use feedback-attention magenta.

Typography

NVIDIA Sans / Consolas only · sentence case · sizes track the type scale · code slides are dark.

Layout

Matches a known pattern · component anatomy is correct · margins, gutters, and column counts within spec.

Logo & hero

Logo lockup not modified · clear space respected · hero composition matches a sanctioned variant.

Icon sourcing

Icons look like NVDAM monoline · in one of the three sanctioned colors · concept vs. product use is appropriate.

Accessibility

WCAG 2.2 AA contrast · focus indicators visible · no color-only signal · text size meets minimum.
