Chart types we use

◐
Doughnut with icon
The Academy's signature chart. Centered concept icon anchors the subject. Use for share/proportion data with 3–5 segments max.
Bar / column
Discrete comparisons. 5–8 bars max. Each bar in one solid color from the chart palette.
Line
Trends over time. Up to 3 series, distinct colors from the chart palette. Axes in gray-200.
93% customer satisfaction
Big number
For a single headline metric. Display 1 size, bold black with a small unit label below.

Don'ts

Don't
Off-system rainbow palette
Use chart-series colors only. No PowerPoint default palette, no Tableau theme, no random tints.
Don't
3D / extruded charts
Flat 2D charts only. No depth, no perspective, no shadows on bars.
Don't
More than 5 segments
Group small slices into "Other" or split across two charts. Beyond five, no one can distinguish the segments.
[chart with
no source or
legend]
Don't
No source citation
Every chart cites its source line. Internal data, public report, telemetry — name it.

Don't
White text on NVIDIA Green
Same WCAG rule as the Color page — black text on green, or use Secondary Green for white text.
