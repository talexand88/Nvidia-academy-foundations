Code uses Consolas on a dark background, always — even inside a light deck. White text on near-black matches every developer terminal. See Code & CLI for sizes and syntax colors.

Weights

Bold
700 · Titles · CTAs
Medium
500 · Subtitles · "semibold"
Regular
400 · Body · default

NVIDIA's "semibold" token = Medium weight (500), not the conventional semibold (600). In PowerPoint, pick NVIDIA Sans Medium.

Sentence case

Do
Course overview
First word capitalized. Everything else lowercase.
Don't
Course Overview
Title case is never used in Academy materials.
Do
Enroll now
Button labels follow the same rule.
Don't
ENROLL NOW
All-caps reserved for short labels and badges only.

Acronyms (AI, GPU, VM, API), proper nouns, and product names keep their capitalization.
