Topologies

How to draw network, system, and infrastructure diagrams — node shapes, connection styles, hierarchies, framing, and labels.

The Anchor Sample — InfiniBand Fat-tree

InfiniBand
A three-tier fat-tree: spine (top), leaf (middle), hosts (bottom). Container framed in cyan with the fabric name in NVIDIA Green at the top-left. Switches use a bidirectional-arrow glyph; hosts use a rack-slot glyph. Connections are thin neutral lines that sit behind the nodes.

Node Spec To Firm Up

  • Exact dimensions in pt for PowerPoint and px for After Effects, per node type.
  • Inside glyphs — the canonical bidirectional-arrow for switches, the rack-slot pattern for hosts. Pull from the icon library where possible.
  • When to use which — switch vs. router vs. NIC, server rack vs. single host, GPU vs. CPU node.
  • Sizing relative to the canvas — what's a "small" topology (≤ 12 nodes), what's a "dense" one.

Connection Styles — Proposed

Lines stay thin, neutral gray, and straight where possible. The point is the structure, not the lines themselves.
  • Default link — 1.2 pt, gray-500, no arrowhead.
  • Highlighted / hot path — 2 pt, NVIDIA Green, no arrowhead.
  • Directional flow — 1.2 pt with single arrowhead at the destination, only when direction matters.
  • Aggregation / bundle — 2 pt parallel pair, when a single line represents multiple links.

Connection Spec To Firm Up

  • Curve vs. straight — when (if ever) to use a curve.

Framing & Labels — Proposed

The InfiniBand sample uses a cyan-bordered rounded rectangle framing the whole topology, with the fabric name in NVIDIA Green at the top-left. Framing is optional — used when the topology represents a named subsystem (fabric, cluster, pod) that needs to be visually bracketed.

Production Spec To Firm Up

  • PowerPoint — build from shapes (not raster). Use the master template's grid for column alignment. Node text in NVIDIA Sans 18 pt; line stroke 1.2 pt.
  • After Effects — animate by tier (spine → leaf → hosts), not by individual node. Hold for 2 seconds after the full topology resolves before transitioning.
