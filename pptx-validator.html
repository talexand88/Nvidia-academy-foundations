PowerPoint Deck Validator

Upload your .pptx. The tool unpacks the deck, checks it against the design system, and returns a compliance report. Catch the wrong canvas size, legacy fonts, white-on-green CTAs, and non-system icons before your deck ships.

Prototype preview. This page shows the intended UI and a sample report. The actual validator runs client-side .pptx parsing (canvas size, fonts, text content, color hex) plus optional server-side icon recognition. Sample report below uses a real Academy deck from the audit.

What the validator checks

  • Canvas & structure — slide dimensions, file size, slide count, masters and layouts present
  • Typography — font families used, font sizes against type scale, line spacing, sentence case, deprecated font detection
  • Color — hex values present, off-system colors, CTA contrast (white-on-green fail), theme color mapping
  • Layout & spacing — line spacing consistency, margin compliance, component anatomy where detectable
  • Icons & imagery — third-party logos retain brand colors, NVDAM-sourced icons vs. custom-drawn (heuristic), hexagon containers limited to product icons
  • Logo — NVIDIA logo present and unmodified, no effects applied, clear space respected, no retired logo variants
