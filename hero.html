Hero section

A full-bleed block that introduces a page or section with the most important content — eyebrow, title, body, and one or two calls to action. Optional background media.

Source. The Hero pattern comes from the Kaizen UI Foundations Hero component. The Academy uses it sparingly — usually as a course intro card or a major section opener.
Academy use is rare. Most slides and web pages don't need a hero. Reserve it for course intros, unit openers, and one-screen marketing pages. For inline title-and-subtitle setups inside a deck, use a normal Page Title from Typography.

When To Use

Use a Hero to prominently display the most important content with visuals and a call to action. The hero is typically the first element users see on a page (below the app bar / header).

Anatomy

Required

  • Eyebrow — context label above the title (e.g., "NVIDIA Academy", course family).
  • Title — the most important content. One or two lines maximum.
  • Body — single-paragraph explanation. Adds context but doesn't restate the title.
  • Primary action — the next step the learner should take.

Optional

  • Background media — image, video, or a generated hero composition. Set mediaTheme to light or dark.
  • Secondary action — outlined button next to the primary. Use only if the second action is genuinely useful (e.g., "Read the docs" alongside "Start course").
  • Tag chips — short metadata (duration, level, format) between title and body.
  • CSS filter — brightness / blur on the media when it's too busy for readable text.

Contrast check. Whatever the media looks like, the hero text must reach WCAG AA contrast against it. If the underlying image makes that impossible, add a darkening overlay (e.g. linear-gradient(rgba(0,0,0,0.45), rgba(0,0,0,0.20))) before placing text.

In PowerPoint & After Effects

PowerPoint course title card. Use the 40 × 22.5 in master, fill with the hero composition image, add a darkening rectangle on top (50% black, full slide), then layer Title at 88 pt bold white, eyebrow at 24 pt bold green above, and a button at 18 pt. This mirrors the "default + dark media" recipe above.

After Effects course opener. 1920 × 1080, same recipe at 76 px title, 16 px eyebrow, 28 px body. Animate by sliding the title and body up from below (220 ms each, 80 ms stagger), then fading in the button group.
